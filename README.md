# Usage
<!-- usage -->
```sh-session
$ npm install -g todocli
$ todocli COMMAND
running command...
$ todocli (-v|--version|version)
todocli/0.0.0 darwin-x64 node-v11.10.1
$ todocli --help [COMMAND]
USAGE
  $ todocli COMMAND
...
```

OPTIONS
